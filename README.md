# 🚀 JyotishConnect (Vedic Caller)

Production-ready astrology voice consulting platform.

## 📱 Tech Stack
- Flutter (Mobile)
- Node.js (Backend)
- PostgreSQL (DB)
- Razorpay (Payments)
- Agora (Voice)

## 🏗️ Structure
```
apps/mobile
backend/api
infra/
```

## ▶️ Run
### Mobile
```
cd apps/mobile
flutter pub get
flutter run
```

### Backend
```
cd backend/api
npm install
node app.js
```

## ✅ Features
- OTP Login
- Wallet system
- Voice calls
- Astrologer marketplace
