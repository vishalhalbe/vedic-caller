# PRODUCT REQUIREMENTS V2 (Full Product)

## Personas
- User (seeker)
- Astrologer (provider)
- Admin

## Core Features
### User
- Auth (OTP + JWT)
- Browse astrologers
- Wallet + payments
- Voice call (Agora)
- Call history
- Ratings & reviews

### Astrologer
- Availability toggle
- Earnings dashboard
- Call acceptance/rejection

### Admin
- Manage users
- Manage astrologers
- Monitor transactions
- Fraud detection

## Non Functional
- 99.9% uptime
- <200ms API latency
- Secure payments

## Missing Gaps Identified
- No astrologer panel
- No call queueing
- No ratings system
- No refunds
- No analytics

## Priority Roadmap
1. Ratings + reviews
2. Call queue system
3. Admin dashboard
4. Analytics
5. Notifications
